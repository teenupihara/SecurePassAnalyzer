# 🔐 SecurePass Analyzer

SecurePass Analyzer is a web-based cybersecurity tool built with Flask that helps users evaluate password security and generate stronger passwords.

---

## ✨ Features

- 🔍 Real-time password strength analysis
- 🔐 Strong password generator
- 🔑 SHA-256 hash generation
- 🔒 bcrypt hash generation
- ⏳ Password crack time estimation
- 📖 Dictionary attack estimation
- 💻 Brute-force attack estimation
- 🚨 Local breach detection
- 📊 Password entropy calculation
- 📱 Responsive user interface

---

## 🛠 Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- zxcvbn

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/teenupihara/SecurePassAnalyzer.git
```

Go into the project

```bash
cd SecurePassAnalyzer
```

Create a virtual environment

```bash
python3 -m venv venv
```

Activate it

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
